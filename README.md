# 🏦 Bank Management System (JavaFX + MVC + Flyway)

A professional-grade desktop banking system built with:

- Java 25
- JavaFX (UI)
- Maven
- MySQL
- Flyway (Database migrations)
- Dotenv (Environment configuration)

This project follows a layered MVC architecture:

UI (JavaFX Controllers)
→ Service Layer (Business Logic)
→ DAO Layer (Database Access)
→ MySQL Database

---

# 📦 Tech Stack

- Java 25
- JavaFX 21
- Maven
- MySQL 8+
- Flyway 10+
- dotenv-java
- jBCrypt (for password hashing)

---

# ⚙️ Prerequisites

Before running the project, ensure:

- Java 25 installed
- Maven installed
- MySQL Server running
- `mysqldump` available in system PATH

Check versions:

```bash
java -version
mvn -version
mysql --version
